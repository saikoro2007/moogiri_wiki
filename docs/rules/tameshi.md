# 大喜利マラソン

- 10回の総得票数で順位を決める大喜利です
- 11回のうち、10回に参加します。参加しない回を !tag で決めます
  - この回は赤が参加しない、みたいな感じです
  - チーム戦ではないので、色で得票数を合計したりはしません
  - どの色が何回目に参加しないかは、!choice11 で決めます
  - 参加しない色の中から、!choiceで出題者を決めます。みなさん、一人一題お持ち寄りください
- 回答時間3分、投票時間5分
- **!msaru3 3m 5m ** で行います
- 1人1回答
- 観戦者も含め、自分以外に絶対評価で投票。各回答に0点から3点をつける
- 色を決めるタイミングで参加締め切り
  - 途中参加はNG
  - 途中抜けは自由です 
- 11回戦終了時、pt圏内で同率が発生した際
  - 5位以上は低い順位から順にサドンデス（ルール後述）で順位を決めます（例: 1位、1位、3位、4位、5位）
  - 6位以下10位以上の同率順位については、同率で10pt差し上げます
  - 例: 1位、1位、3位、4位、5位、5位、5位、8位、9位、9位、9位、9位、13位…のとき
    1. 8位、9位の人は同率10pt
    2. 5位決定戦を行う
    3. 1位決定戦を行う

- サドンデス
  - 回答時間3分、投票時間2分
  - 1人1回答
  - 観戦者のみ、1投票（dsaru）
  - 票数に差がついた時点で、順位を決定（6位以下が確定する場合は、差がついていなくても順位を決定）
  - 差がつかない場合は → 決選投票 → 瞬発大喜利（回答時間1分で、サドンデスと同一ルール）**
