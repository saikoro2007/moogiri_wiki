# サーバーの基本ルール

みんなで大喜利公式Wikiへようこそ！このページでは、サーバー全体の基本的なルールとガイドラインを説明します。

## 1. 相手を尊重しましょう
誰もが快適に過ごせるよう、誹謗中傷や差別的な発言は禁止です。

:::danger [禁止事項]
過度な暴言や差別的発言が確認された場合、警告なしでキック・BANの対象となります。
:::

## 2. スパム行為の禁止
無意味な連投や、不適切なリンクの投稿は控えてください。

:::warning [注意]
外部サーバーへの招待リンクを無断で貼ることは禁止されています。
:::

## 3. チャンネルの使い分け
各チャンネルにはそれぞれの目的があります。目的に合ったチャンネルでの発言をお願いします。
